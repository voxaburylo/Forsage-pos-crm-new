**Тип:** Infrastructure

---

## Що потрібно зробити

1. Встановити Node.js (v18 або новіше)
2. Встановити Git
3. Встановити пакетний менеджер (pnpm або npm)
4. Створити проект в Supabase (https://supabase.com)
5. Отримати Supabase URL (Project Settings - API)
6. Отримати anon key та service_role key
7. Налаштувати .env файл
8. Перевірити що все працює
