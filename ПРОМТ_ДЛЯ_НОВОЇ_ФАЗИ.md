# ПРОМТ ДЛЯ ЗАПУСКУ НОВОЇ ФАЗИ

Вставляй цей текст на початку кожної нової фази.
Замінюй тільки: [НОМЕР ФАЗИ] і [НАЗВА ФАЗИ]

---

## ТЕКСТ ПРОМТУ:

```
Починаємо Фазу [НОМЕР] — [НАЗВА].

Виконай її за строгим протоколом:

КРОК 1 — ЧИТАННЯ (без коду)
Прочитай повністю:
- PLAN_STEPS.md — кроки і список файлів для цієї фази
- ТЗ_ФОРСАЖ_CRM.md — відповідні розділи (endpoints, бізнес-правила, UI вимоги)
- Усі відповідні SPEC_*.md файли якщо є деталі по цій фазі

КРОК 2 — ЧЕКЛИСТ (покажи мені перед кодом)
Склади і покажи повний список:
□ Кожен файл який треба створити (backend + frontend)
□ Кожен endpoint по ТЗ (метод + шлях)
□ Бізнес-правила які треба реалізувати
□ Що оновити в існуючих файлах (index.ts, App.tsx і т.д.)

КРОК 3 — КОД
Напиши всі файли за один прохід.
Для кожного файлу обов'язково перевір:
□ tenant_id присутній в кожному INSERT в БД
□ Гроші — тільки INTEGER копійки, Math.round() при конвертації
□ requireAuth на кожному route (або router.use(requireAuth))
□ Zod валідація на кожному POST / PUT endpoint
□ Soft delete через deleted_at (не фізичне DELETE)
□ Немає console.log (тільки pino logger на сервері)
□ Немає TypeScript any без коментаря
□ Українська мова в UI (labels, errors, buttons)
□ Error format: { error: { code, message, status } }
□ Всі endpoint з ТЗ реалізовані (не пропущено жодного)
□ Бізнес-правила перевірені логічно (не тільки синтаксично)

КРОК 4 — САМОАУДИТ (до того як показати мені)
Пройдись по КОЖНОМУ написаному файлу і звір з чеклистом.
Потім запусти:
- npx tsc --noEmit (server) — результат: 0 помилок
- npx tsc --noEmit (apps/web) — результат: 0 помилок
- npx vite build — результат: ✓ built

КРОК 5 — ЛОГІЧНА ПЕРЕВІРКА
Перевір руками (читаючи код):
□ Якщо зміна не відкрита — продаж неможливий?
□ Якщо борг — клієнт обов'язковий?
□ Після продажу — qty_on_hand зменшується?
□ Всі нові routes зареєстровані в index.ts?
□ Всі нові сторінки додані в App.tsx?
□ tenant_id є в усіх INSERT операціях?
□ Тестовий user має потрібну роль для тестування?

КРОК 6 — ЗВІТ
Тільки після успішного проходження всіх кроків покажи:

✅ ФАЗА [N] ЗАВЕРШЕНА

Файли створені:
- [перелік всіх файлів]

Endpoints реалізовані:
- [перелік всіх endpoints]

Самоаудит:
- TypeScript server: 0 помилок
- TypeScript frontend: 0 помилок
- Vite build: ✓
- tenant_id в INSERT: ✅
- Zod валідація: ✅
- requireAuth: ✅
- Бізнес-логіка: ✅

git commit: [хеш] — [повідомлення]
```
