# SPEC: Дизайн-система "Форсаж"

## Бренд

| Параметр | Значення |
|---|---|
| Назва | Форсаж |
| Логотип | Нахилена літера F (курсив, +15°) |
| Акцентний колір | `#FFD000` (жовтий) |
| Темний фон POS | `#1A1A1A` |
| Світлий фон адмінки | `#F5F5F5` |
| Шрифт | Inter або Roboto (кирилиця) |

---

## Кольорова палітра

### Основні кольори
```
Акцент (жовтий):   #FFD000  — кнопки дій, логотип, підсвічування
Фон темний:        #1A1A1A  — POS, нічний режим
Фон середній:      #2C2C2C  — картки в POS
Поверхня:          #FFFFFF  — адмінка
Фон сторінки:      #F5F5F5  — загальний фон
```

### Семантичні кольори
```
Успіх / є в наявності:    #22C55E  (зелений)
Попередження / мало:       #F59E0B  (помаранчевий)
Небезпека / немає / борг: #EF4444  (червоний)
Інфо / нейтральний:       #3B82F6  (синій)
Вимкнено / архів:         #9CA3AF  (сірий)
```

### Статуси залишків товару
```
Є в наявності (qty > min_qty)  → зелена крапка  #22C55E
Мало (qty <= min_qty > 0)      → жовта крапка   #F59E0B
Немає (qty = 0)                → червона крапка #EF4444
Під замовлення                 → синя крапка    #3B82F6
Архів / неактивний             → сіра крапка    #9CA3AF
```

## Картка товару

```
Містить:
- Фото: 1:1 ratio, 120px у списку / 240px у картці, lazy load
- Назва: max 2 рядки, overflow ellipsis
- Артикул: моноширинний шрифт, клік = копіювати
- Закупівельна ціна: видима тільки ролям Owner/Admin
- Ціна продажу: жирна, акцентна
- Статус залишку: кольорова крапка + кількість
- Швидкі дії: [Додати в чек] [Детальніше]
```

## Логотип

```
Форма: літера F нахилена +15° (правий нахил)
Шрифт: жирний геометричний sans-serif без засічок
Колір: #FFD000 на темному / #1A1A1A на жовтому

Розміри:
  favicon:          32×32px
  sidebar згорнутий:32×32px
  sidebar розгорнутий: іконка + текст "Форсаж"
  документи:        мінімум 40px висота

Заборонено: розтягувати, змінювати кут, ставити на кольоровий фон
```

## Анімації

## Toast-повідомлення

```
Успіх:        зелений, іконка Check, 3с

Попередження: жовтий, іконка Alert, 4с
Інфо:         синій, іконка Info, 3с

Позиція: правий нижній кут (не перекриває кнопки POS)
```

## Друковані документи

```
Шрифт: Arial (гарантована підтримка на всіх принтерах)
Чек: 58мм або 80мм (thermal printer)
Накладна/КП: A4, поля 15мм

Обов'язкові елементи на кожному документі:
- Логотип Форсаж (чорно-білий для thermal)
- Назва магазину і адреса
- Дата, час, номер документа
- Підпис відповідального (для накладних)
```

## Адаптивність

```
Desktop ≥1280px:  повний sidebar + двоколонковий POS
Tablet 768-1279px: sidebar collapsed + POS одна колонка
Mobile <768px:    тільки перегляд, пошук, замовлення (каса недоступна)
POS-термінал:     фіксований мінімум 1024×768px, завжди темна тема
```
